Archive Contents List

The Contents view lists the contents of the currently opened ZIP archive. It lists a few pieces of information for each file,

  • The file name.
  • The uncompressed size. This is the actual size the file will take up on disk when extracted.
  • The compressed size in the archive.
  • The compression ratio.
  • The full path and file name. This is the full path the file will be placed in when extracted (relative to the extraction location).
